dataset_1M_speed2_accel1 CSV 컬럼 설명
(merged.csv 및 shape_dataset/ 안의 개별 에피소드 CSV 공통)

dataset_1M과 동일한 파이프라인으로 수집했고, 차이는 max_accel만 2.0 -> 1.0m/s^2로 낮춘 것
(max_speed는 2.0m/s 그대로, --att_d_gain_scale 0.3도 동일하게 적용).

각 행 = 드론 시뮬레이션의 한 컨트롤 스텝(0.01초)의 기록.
이 데이터셋은 자세 제어 D게인을 0.3배로 낮춘 설정(--att_d_gain_scale 0.3)으로 수집됨
(velocity-only 제어 방식에서 나타나는 롤/피치 진동을 완화하기 위함, 자세한 이유는 대화 기록 참고).

[시간/인덱스]
- episode_id   : 에피소드 번호 (episode index) — merged.csv에만 있음, 0부터 시작, 몇 번째 비행인지
- step          : 스텝 번호 (step index) — 해당 에피소드 안에서 0부터 시작
- t             : 경과 시간 (time in seconds) — 초 단위

[state: 드론의 실제 상태 — 시뮬레이션 물리가 계산한 진짜 값]
- pos_x, pos_y, pos_z       : 위치 (position x/y/z) — 미터(m)
- vel_x, vel_y, vel_z       : 속도 (velocity x/y/z) — 미터/초(m/s)
- roll, pitch, yaw          : 자세각 (roll angle, pitch angle, yaw angle) — 라디안(rad) 단위 주의
- ang_vel_x, ang_vel_y, ang_vel_z : 각속도 (angular velocity x/y/z) — 라디안/초(rad/s)

[action: 드론에게 내린 목표 속도 명령 — 실제 속도(vel_*)와는 다른 값]
- action_vx, action_vy, action_vz : 목표 속도 (action velocity x/y/z) — m/s
- action_yaw_rate            : 목표 요(yaw) 각속도 (action yaw rate) — rad/s, 이 데이터셋에서는 항상 0

[참고용 목표 지점 — 실제 도달 위치가 아니라 "조준한" 경로 기준점]
- target_pos_x, target_pos_y, target_pos_z : 목표 위치 (target position x/y/z) — m
  (pos_x,y,z와의 차이가 곧 추종 오차. 완벽히 같지 않은 게 정상)

[학습용 라벨]
- reward  : 보상 (reward) = -(pos - target_pos)의 거리 제곱. 0이 최선(완벽 추종), 음수로 갈수록 오차가 큼
- done    : 에피소드 종료 여부 (is done) — 해당 에피소드의 마지막 행에서만 True, 나머지는 False

[에피소드(비행) 메타데이터 — 한 에피소드 안에서는 모든 행이 동일한 값]
- shape          : 도형 종류 (shape type) — triangle / square / pentagon / circle 중 하나
- center_x, center_y, center_z : 도형 중심 위치 (shape center x/y/z) — m
- start_yaw_deg   : 도형 시작 회전각 (shape start yaw) — **도(degree)** 단위 (roll/pitch/yaw와 단위 다름 주의)
- tilt_deg        : 도형 평면 기울기 각도 (shape tilt angle) — **도(degree)** 단위
- tilt_axis_deg   : 기울어진 축의 방향각 (shape tilt axis angle) — **도(degree)** 단위
- max_speed       : 이 에피소드에 적용된 최고 속도 (episode max speed) — m/s. 이 데이터셋은 전 에피소드 2.0m/s로 고정
- max_accel       : 이 에피소드에 적용된 최고 가속도 (episode max acceleration) — m/s^2. 이 데이터셋은 전 에피소드 1.0m/s^2로 고정

※ 단위 주의: roll/pitch/yaw/ang_vel_*는 라디안(rad)이고, start_yaw_deg/tilt_deg/tilt_axis_deg는
  이름 그대로 도(degree)입니다. 서로 다른 컬럼이니 혼동하지 마세요.
